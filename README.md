**Getting Started**

First, run the development server:


**Using yarn (recommended)**

```markdown
// install
yarn install

// start dev server
yarn run dev

// build
yarn build

//work on
localhost://8081
```



**Using docker container**

```markdown
// install & start
make start

//work on
localhost://9802:8081
```
